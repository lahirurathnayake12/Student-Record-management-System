
This is a simple student database software built for the Data Structures and Algorithms assignment.

Data Structures Used:
- Singly Linked List: To save student records dynamically.
- Queue: To manage student waiting lines fairly (FIFO).
- Stack: To save deleted data so users can Undo mistakes.

Main Algorithms Used:
Linear Search: To search for students using their ID.
 Queue Workflow: To add and remove students from the line.
 Stack Workflow: To push deleted files and pop them back.

How to Setup and Run the Project:
1. Download all the java files to a folder on your computer.
2. Open Command Prompt (CMD) inside that specific folder.
3. Type "javac *.java" and press Enter to compile the code.
4. Type "java main" and press Enter to start the system.
5. Choose numbers from the console menu to test the features.
